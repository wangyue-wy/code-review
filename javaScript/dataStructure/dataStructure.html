# 所有form表单应该加上Layout布局

线损的form表单在改变屏幕大小时，由于没用Layout布局，造成布局出错（除了form表单，当有一个盒子里面再一行存在几个盒子时，应该也要使用Layout布局）

# 表格的切换应该用数据驱动

页面具有表格切换功能时，尽量不要有几个表格的切换就写多少个表格demo,或者有几个表格就封装几个组件。而是用数据驱动来改变表格的切换

// 在methods中定义一个方法，返回的在不同条件下对应的值
/**
 * @param id 对应的条件
 * @param list 需要赋值的数组
*/
getSeries (id, list) {
    let detail = this.seriesDataAll.find(v => v.id === id)?.detail
    list.map((item, index) => {
        detail[index].data = item
    })
    return detail
}
